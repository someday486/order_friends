# Kickoff Summary (2026-01-13)

본 문서는 Order Friends 프로젝트의 1차 킥오프 미팅에 대한
**사실 기반 요약(Summary)** 이다.
회의에서 실제로 공유·합의된 범위만 기록하며,
이후의 설계·구현 내용은 별도 문서에서 관리한다.

## 2. Meeting Purpose

## 3. Topics Discussed
- 서비스는 **여러 브랜드와 매장을 동시에 운영할 수 있는 구조**가 필요하다는 점
- 사용자, 브랜드, 매장 간의 관계를 **명확한 구조로 정리할 필요성**
- 초기 단계에서는 **구조/설계 정리가 우선**이라는 점

> 세부 기능, 일정, 기술 스택에 대한 확정 논의는 하지 않음

---

## 4. Agreed Points
- 프로젝트를 진행하기로 결정
- 초기에는 **데이터 구조와 기본 개념 정리부터 진행**하기로 함
- Brand / Branch 단위로 나뉘는 구조가 필요하다는 데 공감
